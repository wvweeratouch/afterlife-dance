ARCHITECTURE SUMMARY

Thread 1 — Audio: Tone.Transport. All loops (pad, melody, mel2, sparkle, heart, gibberish, sub, drums, retrig, grace). Never blocks.
Thread 2 — Detection: setInterval(200ms). Camera 320×240 → offscreen 80×60. Blob clustering. MediaPipe every other detect call. Calls triggerFromBlobs() at end.
Thread 3 — Draw: requestAnimationFrame. Stars, blob glow/trails, skeleton overlay. No audio, no detection.

Auto-detection: Scene switch → SC[n].det checked → 'off' skips detection, 'blob' runs blob only, 'pose+blob' also runs MediaPipe.
Light mode: blob threshold 80+(100-sens)*0.8 | Dark mode: threshold 25+(100-sens)*0.3
Transitions: pad.volume.rampTo(), mel.volume.rampTo(), rev.wet.rampTo() — all smooth via Tone.js
